Math Typer

Format normal text into math symbols in real time!

Write whatever you want in the left panel (top panel if you're on mobile). The program recognizes some set commands and converts them into mathematical symbols.

Recognized commands as of now:

./ returns a division symbol (÷)

.r returns a root symbol (√)

.3r returns a cube root symbol (∛)

.4r returns a fourth root symbol (∜)

..c.. returns a congruency symbol (≅)

tri returns a triangle symbol (∆). Note that there should be a space after 'tri'

This is a work-in-progress. More commands will be added in the future.
